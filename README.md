# 🌐 Personal Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript 
to showcase my skills, projects, and contact information.

## 🔗 Live Demo
[View Portfolio](https://kashish-kumar-angrish.github.io/Portfolio-website/)

## 📸 Preview
![Portfolio Preview](images/preview.png)

## ✨ Features
- Responsive design that works on all devices
- Smooth scroll navigation
- Animated UI using GSAP
- Projects showcase section
- Contact form
- Clean and modern UI

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- GSAP (Animations)

## 🚀 Getting Started

To run this project locally:

1. Clone the repository
   git clone https://github.com/kashish-kumar-angrish/Portfolio-website.git

2. Open the project folder
   cd Portfolio-website

3. Open index.html in your browser

No installation or dependencies required.
